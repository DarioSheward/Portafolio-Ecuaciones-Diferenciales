# Portafolio - Ecuaiones Difrenciales (2026)

EL formato de este portafolio esta inspirado por la estrategia de trabajo del curso de Física Computacional II (2024) del Profesor Roberto Navarro.

- El documento principal es [portafolio.tex](portafolio.tex).
- Para cada evidencia, deberá crear un documento de latex que será alojado en la carpeta [tex/](tex/), con imágenes en la subcarpeta [img/](img/).
- Si usted crea software (pogramas de python), estas deben estar alojadas en la carpeta [src](src/).
---
La intención de este es portafolio es ir registrando distintos métodos para resolver Ecuaciones Diferenciales de diversos tipos.
